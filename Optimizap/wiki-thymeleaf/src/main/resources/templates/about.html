Propósito

Esta wiki organiza y presenta información de forma estructurada para simular un portal de documentación técnica.

Alcance del taller

Esta aplicación está orientada a demostrar buenas prácticas de arquitectura web server-side, uso de vistas dinámicas y reutilización de componentes de interfaz con Thymeleaf.

  • Renderizado dinámico de contenido con Thymeleaf
  • Navegación coherente entre secciones
  • Uso de fragmentos reutilizables (header, footer, menú)
  • Formulario de contacto con validaciones en JavaScript
  • Despliegue en contenedor Docker con instrucciones de ejecución
